10.3 error handling

The window.lib.getPersons() function is available in the script. It must receive a callback as a parameter. After a short delay, the callback will be executed with two parameters: the first contains an error (if there is one, otherwise null ), the second an array of people.

When clicking on the button, call the getPersons function and display the result obtained in the console. Warning: the getPersons function has plus or minus one in two chance of generating an error. If that is the case, uses console.error () to display it.
