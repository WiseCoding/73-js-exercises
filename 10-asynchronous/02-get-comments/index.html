10.2 loading articles and comments

You must have noticed, in the previous exercise, that the articles received via the window.lib.getPosts function included a property id.

The window.lib.getComments () function is available in the script. It should be given an item id as the first parameter, so as a second parameter callback. After a short delay, the callback will be executed with two parameters: the first contains an error (if there is any one, otherwise null ), the second an array of article comments.

When you click on the button, calls the getPosts function. For each item retrieved, calls the getComments function and adds the comments obtained in a comments property of the article. Display articles in the console (the getPosts and getComments functions never return error).
