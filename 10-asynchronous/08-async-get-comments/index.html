10.8 loading articles and comments (async/await)

You must have noticed, in the previous exercise, that the articles received via the window.lib.getPosts function included a property id .

The window.lib.getComments () function is available in the script. It should receive an item id as a parameter, and return a Promise which will be resolved, after a short delay, with the table of comments on the article.

When you click on the button, calls the getPosts function. For each item retrieved, calls the getComments function and adds the comments obtained in a comments property of the article. Display articles in the console (the getPosts and getComments functions will always be resolved without error).

Note : for this exercise, use the syntax async/await.
